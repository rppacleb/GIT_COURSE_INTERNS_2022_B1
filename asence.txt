Basic steps on pushing and pulling a changes and steps when a conflict occur
    STEP 1: git add .
    STEP 2: git commit -m ''
    STEP 3: git pull (so that we can get updates to other devs)
    STEP 4: fix conflict if occur (decide changes with your team)
    STEP 5: git add .
    STEP 6: git commit -m ''
    Step 7: git push

Fix conflict talk with other devs if what the team will accept incoming/current changes etc.
